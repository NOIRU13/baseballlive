# 要件定義：スリーアウトチェンジのアニメーション

## 概要

野球の試合において攻守交代（チェンジ）は重要な演出ポイントである。現状の簡素な表示を、よりリッチでプレミアム感のあるアニメーションに置き換える。

## 要件

1.  **トリガー**: スコア管理者が「アウト」「三振」または「凡退系の打席結果」を入力し、アウトカウントが3になった瞬間。
2.  **演出内容**:
    - 画面中央に大きく「3 OUTS CHANGE」等の文字が表示される。
    - フォントは高級感のあるもの（OutfitやInterなど）を使用。
    - アニメーションは「ゆったり、ふんわり」とした動き（Ease-in-out、スケーリング、フェード）。
    - 4〜5秒間表示され、視聴者にチェンジを明確に伝える。
3.  **複数ページ対応**:
    - `index.html` (表示用)
    - `admin-simple.html` (管理用、確認のため)
    - その他 `broadcastResultEvent` を受信する全てのページ。
