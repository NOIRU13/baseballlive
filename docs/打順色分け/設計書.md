# 設計書

## 変更ファイル

- js/ui/render.js: positions に応じて適切な CSS クラス (pos-catcher など) を付与するロジックを追加。
- css/player.css: 付与されたクラスに対するスタイルを定義。

## CSS クラス設計

.player-pos をベースに背景色を追加。

- .pos-pitcher: Red Gradient
- .pos-catcher: Blue Gradient
- .pos-infielder: Yellow Gradient (Dark Text)
- .pos-outfielder: Green Gradient
- .pos-dh: Purple Gradient

## UI ロジック

- getPositionClass(pos) ヘルパー関数を作成し、守備位置文字列からクラス名を判定。
- updateLineupDisplay 関数内でポジション要素生成時にこのクラスを追加。
