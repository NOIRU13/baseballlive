# 管理画面改修 要件定義

## 目的

`admin-control.html` からチーム選択機能を削除し、チーム・ラインナップ管理を `admin-lineup.html` に集約する。

## 現状

- `admin-control.html`: 試合進行操作とチーム選択機能が混在している可能性。
- `admin-lineup.html`: ラインナップ管理機能が存在するはず。

## 改修内容

1. `admin-control.html`
   - チーム選択（先攻・後攻）の UI を削除する。
   - 必要なデータ（選択済みチームIDなど）は `admin-lineup.html` で設定されたもの、または現在の試合状態から参照するように修正する（API経由など）。

2. `admin-lineup.html`
   - チーム選択からラインナップ設定までを一貫して行えることを確認する。
   - もし不足があれば機能を追加する（今回は「完全管理」とのことなので、この画面だけで完結させる）。
