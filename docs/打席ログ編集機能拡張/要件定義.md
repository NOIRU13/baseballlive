# 要件定義：打席ログ編集機能拡張

## 1. 目的

試合中に入力された各打席の結果、打点、投手の失点・自責点を後から一覧で確認・修正できるようにし、スコア記録の正確性を向上させる。

## 2. 追加機能

- **打席詳細ログの保存**: 全ての打席結果入力時に、詳細データ（イニング、打退、打者、投手、結果コード、打点、失点、自責点）を状態として保持する。
- **ログ一覧表示**: 「試合管理」タブにおいて、これまでの全打席ログをテーブル形式で表示する。
- **インライン編集**: 各打席の結果、打点、失点、自責点をその場で編集・保存できるようにする。
- **統計データの自動更新**: ログが編集された際、打者の当日累計打点や投手の失点・自責点、打席履歴を自動で再計算・更新する。

## 3. 対象データ

- 結果（単打、本塁打、ゴロ等）
- 打点（RBI）
- 失点（Runs Allowed - 投手）
- 自責点（Earned Runs - 投手）
