# 要件定義書 - バッター表示機能

## 1. 概要

野球スコアボードシステムにおいて、現在の打者情報（名前と当日の打撃成績）をリアルタイムで表示する専用画面（`Batter.html`）を作成する。
OBS等の配信ソフトでの利用を想定し、視認性の高いデザインとする。

## 2. 機能要件

### 2.1 表示機能

- **現在の打者名表示**:
  - 管理画面で設定・操作されている現在の打者名を大きく表示する。
- **打撃成績表示**:
  - その打者の当日の全打席結果（ヒット、アウト、四死球など）をバッジ形式で表示する。
  - 左から右へ、時系列順に表示する。
- **デザイン**:
  - 既存のスコアボードのデザインテーマ（クリムゾンレッド、ゴールド）を踏襲する。
  - 背景は透過またはOBSで透過しやすい色とする（今回は `style.css` の `overlay-mode` を活用）。

### 2.2 データ連携

- **リアルタイム同期**:
  - 管理画面（`admin.html`）での操作が即座に反映されること。
  - 既存の `localStorage` または `BroadcastChannel` による同期機構を利用する。

## 3. 非機能要件

- **レスポンシブ**:
  - 特定の解像度（例: 1920x1080の一部）での表示を想定するが、ウィンドウサイズに合わせて適切にリサイズされること。
- **保守性**:
  - 既存の `script.js`、`style.css` を可能な限り流用し、重複コードを避ける。
